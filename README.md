# op_labs

Лабораторные работы по ОП

## Как сделать коммит:

1) Добавить файлы. Сделать коммит. - Подходит, если есть новые файлы

```bash
git add <перечислить файлы>
git commit -m "сообщение для коммита"
```

2) Сделать коммит сразу с добавлением файлов - Используется, когда нет новых файлов,
а есть только изменённые.

```bash
git commit -a -m "сообщение для коммита"
```

## Типичная последовательность действий:

1) git pull
2) работаем, делаем коммиты
3) git push
